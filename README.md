# Transform raw file
Programm gets raw hex file and tranforms them into csv txt or xlsx
# File format
## Input 
- raw hex
## Output 
- csv
- txt
- xlsx
# Usage Example
```sh
./open.bin --file raw  --format csv
./open.py --file raw --format csv
./open.py --dir ./data raw --format csv
./open.bin --dir ./data raw --format csv
```
